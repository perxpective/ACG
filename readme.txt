To Prepare
1. Unzip the zip file Implementation
2. Open the unzipped folder on Visual Studio Code
3. Python 3.10 or later is required to run the programs
4. Open two new terminals and run the two commands respectively:
	python ./source/server/server.py
	python ./source/client/client.py

5. Make sure to run the server program first before running the client program
6. Client will display its certificate and will send images over to the server for processing
7. On the ./source/server/data folder, you will see an increasing number of images.
